# MarginKenya

Monorepo structure:

- `frontend/` - React + Vite client app
- `backend/` - Backend service (to be implemented)

## Frontend

From the project root:

```powershell
cd frontend
npm install
npm run dev
```

Build frontend:

```powershell
cd frontend
npm run build
npm run preview
```

## Backend

Backend folder exists at `backend/` and is ready for implementation/deployment setup.

## Suggested Deployment Layout

- Deploy frontend from `frontend/`
- Deploy backend from `backend/`
